### RELEVE_BANCAIRE
{{releve_bancaire}}

Ta mission :
Effectuer un rapprochement bancaire approximatif en utilisant :
- la similarité fournisseur,
- l’écart de montant,
- la proximité de date.

Important :
- Le champ "invoice_type" fourni dans la facture indique s’il s’agit d’une facture envoyée ou reçue.
- Pour une facture reçue : les montants bancaires doivent être comparés en valeur absolue.
- Le signe négatif d'une transaction n’est jamais une différence.
- La devise ne doit pas être prise en compte.

Une ligne est correspondante si :
- la similarité fournisseur est suffisante (≥ 0.60),
ET
- le montant ou la date sont rapprochés.

Pour chaque ligne correspondante :
- liste les différences constatées,
- indique les écarts chiffrés dans "details_differences",
- calcule la similarité fournisseur,
- calcule le niveau de confiance.

Retourne uniquement le JSON demandé dans le contexte système.
